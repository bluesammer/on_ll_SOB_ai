 RESET_BASELINE_ON_CHANGE, set 1 or 0
Value 0
 Baseline stays fixed.
 If the file stays changed, you keep getting change alerts every run.
Value 1
 After the script processes a change, it copies current into baseline for the affected file.
 Next run treats the new version as the benchmark.
 You stop getting repeated alerts for the same change.
